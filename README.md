# OpencvTutorial
Exploring different methods available in Opencv in Python.

topics:
* Colorize Black and White Photos using a Caffe Model
* EasyOCR
* Face Recognition (Voila-Jones algorithm)
* GrabCut Algorithm
* Neural Style Transfer
* Object Detection
  * YOLOv3
  * Single Shot Detectors (SSDs)
* Object Tracking
